# Codex 運用

このリポジトリの Codex 運用ルールとスキルの実体は、同階層にある `portfolio/.codex` を参照する。

## ディレクトリ前提

```text
dev/
  portfolio/
    .codex/
  bookman_backend/
    .codex/
  bookman_nextjs/
```

## 参照先

- Rules: `../portfolio/.codex/rules/`
- Skills: `../portfolio/.codex/skills/`

ルールやスキルを更新する場合は、原則として `portfolio/.codex` 側を更新し、このリポジトリには重複コピーを置かない。
